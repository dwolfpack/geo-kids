# Glossary (English)

## AI

Artificial Intelligence — computer software that can handle tasks that normally need human thinking, like writing, answering questions, or summarizing. In this course, "AI" usually means AI chat tools such as ChatGPT, Claude, and Gemini.

## Model

The "brain" behind an AI tool — a program trained on enormous amounts of text that predicts, one word at a time, what a good response looks like.

## Chatbot

A program you talk to by typing messages, and it types back. Modern AI chatbots are powered by large language models.

## Prompt

The message you type to an AI. A clear, specific prompt usually gets a much better answer than a vague one.

## Hallucination

When an AI states something false as if it were true — made-up facts, sources, or details — while sounding completely confident.
